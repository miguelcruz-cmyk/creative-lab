# Automations

Ready-to-use [Cursor Automation](https://docs.cursor.com/automations) prompts for
performance marketers. Each file is a self-contained prompt you can paste into a
new Automation (or run on demand in chat). They use **your own** connected
accounts and data — no setup beyond connecting the relevant MCP.

| Automation | What it does | Needs |
| --- | --- | --- |
| [Keyword Gap Finder](./keyword-gap-finder.md) | Finds Google Ads search terms & product launches driving clicks with no matching active keyword, and returns a prioritized add plan. | Google Ads MCP (read) |

## How to use an automation

1. Open the file and copy the **Instructions** prompt.
2. In Cursor, create a new Automation, paste it into Instructions, and set the
   trigger + delivery destination (Slack, email, report).
3. Or just paste it into a chat to run it once.

Want to add your own? Drop a new markdown file here following the same shape
(what it does → setup table → Instructions prompt → output shape → guardrails)
and add a row to the table above.
