# Skills

[Cursor Skills](https://docs.cursor.com/skills) bundled with Creative Lab. Cursor
auto-discovers skills in this folder when you open the repo, so the agent can use
them on demand — just describe what you want in plain language.

| Skill | Use it when you want to… | Reads |
| --- | --- | --- |
| [performance-anomaly-tracker](./performance-anomaly-tracker/SKILL.md) | Detect sudden changes in paid performance (spend spikes, CPA/ROAS drift, creative fatigue, conversion breaks) and get a prioritized alert digest. | Creative Lab API and/or ad-platform MCPs |

Using another agent (e.g. Claude)? Copy the skill folder into that tool's skills
directory (for example `.claude/skills/`).
